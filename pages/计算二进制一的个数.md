- ## 自行实现
	- ### Go
		- ```go
		  func count_ones(num int) int {
		      c := 0
		      for num != 0 {
		          if num & 1 == 1 {
		              c++
		          }
		          
		          num >>= 1
		      }
		      return c
		  }
		  ```
