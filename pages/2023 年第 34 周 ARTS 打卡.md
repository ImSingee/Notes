# Tip - 本周学到的技术技巧
	- Golang 判断接口值是否为空（不利用反射，精准判断）
	- ```go
	  type eface struct {
	   rtype unsafe.Pointer
	   data  unsafe.Pointer
	  }
	  
	  func unpackEFace(obj interface{}) *eface {
	   return (*eface)(unsafe.Pointer(&obj))
	  }
	  
	  func IsNil(obj interface{}) bool {
	   if obj == nil {
	    return true
	   }
	   return unpackEFace(obj).data == nil
	  }
	  ```
