- # 题目
	- 给定两个整数 `n` 和 `k`，返回范围 `[1, n]` 中所有可能的 `k` 个数的组合。
	- 你可以按 **任何顺序** 返回答案。
	-
	- **示例 1：**
		- **输入：**n = 4, k = 2
		- **输出：**
		- [
		- [2,4],
		- [3,4],
		- [2,3],
		- [1,2],
		- [1,3],
		- [1,4],
		- ]
		-
	- **示例 2：**
		- **输入：**n = 1, k = 1
		- **输出：**[[1]]
		-
		-
		-
	- **提示：**
		- `1 <= n <= 20`
		- `1 <= k <= n`
- # 解
	- ```go
	  var arr []int
	  var add func([]int)
	  var n int
	  var k int
	  
	  
	  func combine(N int, K int) [][]int {
	      n = N
	      k = K
	      
	      arr = slice(n)
	      
	      var result [][]int
	      add = func(x []int) {
	          result = append(result, x)
	      }
	      
	      dfs(nil, 0, 0)
	      
	      return result
	  }
	  
	  func dfs(prefix []int, i int, c int) {
	      if c == k {
	          add(append([]int{}, prefix...))
	          return
	      }
	      if i == n {
	          return
	      }
	      
	      dfs(prefix, i+1, c)
	      dfs(append(prefix, arr[i]), i+1, c+1)
	  }
	  
	  func slice(n int) []int {
	      arr := make([]int, n)
	      for i := range arr {
	          arr[i] = i+1
	      }
	      return arr
	  }
	  ```
