- # 题目
  	给你一个整数 <code>n</code> ，统计并返回各位数字都不同的数字 <code>x</code> 的个数，其中 <code>0 &lt;= x &lt; 10<sup>n</sup></code><sup>&nbsp;</sup>。
  	
  	- 
  	- **示例 1：**
  - **输入：**n = 2
  - **输出：**91
  - **解释：**答案应为除去 `11、22、33、44、55、66、77、88、99` 外，在 0 ≤ x < 100 范围内的所有数字。 
  	- **示例 2：**
  - **输入：**n = 0
  - **输出：**1
