- # 题目
	- 给定整数 `n` ，返回 *所有小于非负整数 `n` 的质数的数量* 。
	-
	- **示例 1：**
		- **输入：**n = 10
		- **输出：**4
		- **解释：**小于 10 的质数一共有 4 个, 它们是 2, 3, 5, 7 。
	- **示例 2：**
		- **输入：**n = 0
		- **输出：**0
	- **示例 3：**
		- **输入：**n = 1
		- **输出**：0
		-
	- **提示：**
		- `0 <= n <= 5 * 106`
- # 解
	- ```go
	  var N = 5000000
	  
	  var nums = make([]bool, N+1)
	  var count = make([]int, N+1)
	  
	  func init() {
	      for i := 2; i <= N; i++ {
	          if !nums[i] { // i 是质数
	              for x := 2 * i; x <= N; x += i {
	                  nums[x] = true
	              }
	          }
	      }
	      
	      x := 0
	      for i := 2; i <= N; i++ {
	          count[i] = x
	          
	          if !nums[i] {
	              x++
	          }   
	      }
	  }
	  
	  
	  func countPrimes(n int) int {
	      return count[n]
	  }
	  ```
