- 一旦涉及到子序列和最值，那几乎可以肯定，考察的是动态规划技巧，时间复杂度一般都是 O (n^2)。
-
- ## 思路一
	- 第一种思路模板是一个一维的 dp 数组
	- ```java
	  int n = array.length;
	  int[] dp = new int[n];
	  
	  for (int i = 1; i < n; i++) {
	      for (int j = 0; j < i; j++) {
	          dp[i] = 最值(dp[i], dp[j] + ...)
	      }
	  }
	  ```
