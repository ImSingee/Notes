- 不要用限量版优惠券。
  
  在预售开始的前一天晚上，我给候补用户发了一封邮件，里面有一张20美元的优惠券。它被限制在前50个使用它的人。
  
  优惠券在90秒内就用完了。
  
  电子邮件发送需要超过90秒才能完成，所以有些人在收到电子邮件的那一刻就点击了链接，结果发现优惠券代码不起作用。很多人给我发邮件，告诉我优惠券坏了。
